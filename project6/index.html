Origin & Description

Dog Paddle is a game where the player must guide the paddle by running in a certain direction in real life by using the two game ubttons in order to prevent the dog from falling out of frame and help the dog catch the tennis ball. Every single time that the dog bounces, it starts moving a little bit faster and therefore the difficulty of the game automatically increases through.

Process

I created this game by using JavaScript and a MakeyMakey Box in order to give an immersive feel to a classic ball guiding game. I created my physical interface by creating a circuit with household materials: two kitchen bowls and aluminum foil.

Skills/Softwares/Languages
