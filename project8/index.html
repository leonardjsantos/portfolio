Origin & Description

In the Diver Mini-Game, you play a young diver who must charge up (with your running power) in order to jump into the pool as far as he can. This running power is generated by running on a homemade game pad that I connected to my laptop using a MakeyMakey Box.

Process

I came up with this idea after using some physical controllers during class, in which my group used the Makey Makey box to play with another game I created. I've never done anything like that before, so I wanted to challenge myself by creating an experience that seemed to warrant physical activity so that I could use my new knowledge of physical controllers. Below, you can see pictures of my group and me using physical controllers for the first time. This was the result of that. I thought about this in relation to how I usually had to build up the nerve in order to jump off of the high diving board at the swimming pool when my heart was beating FAST.
