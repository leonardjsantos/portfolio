Origin & Description

Wave-A-Wand is a variation of the popular Whack-A-Mole game, where the player must hit the images of Voldemort when he pops up. Using the physical interface, the user must hit the right button where he pops up in order to gain a point, and if the user misses, the user loses a point.

Process

This game was created using JavaScript and a Makey Makey Box.
